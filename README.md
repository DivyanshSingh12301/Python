# Python Data Analysis Learning

A collection of Python scripts and notebooks documenting my learning journey with **Python**, **NumPy**, and **pandas**. 

Each project applies core concepts learned from [Automate the Boring Stuff with Python](https://automatetheboringstuff.com/) and [Keith Galli's YouTube channel](https://www.youtube.com/c/KeithGalli) to real-world datasets.

## What's Inside

- **Python fundamentals** - Core concepts, loops, functions, file handling
- **NumPy practice** - Array operations, vectorization, mathematical computations
- **Pandas exploration** - Data loading, cleaning, manipulation, basic analysis
- **Real-world datasets** - Applied learning on actual data (Olympics, coffee shop sales, etc.)

## Datasets Used

- Olympics data
- Coffee shop sales daata

## Technologies

- Python 3.x
- NumPy
- Pandas
- VS code
- Git/Github

## How to Use

1. Clone the repository
2. Install dependencies: `pip install numpy pandas`
3. Run scripts or open notebooks to see the code and outputs

## Learning Path

This repo tracks my progress as I move through:
1. Python fundamentals and automation
2. NumPy for numerical computing
3. Pandas for data manipulation and analysis

Each file represents applying these concepts to real data.
